# ربات تراکنش روی شبکه [morph l2](https://x.com/MorphL2) ( تست نت )

## نحوه اجرا
1. دانلود کنید
```
git clone https://github.com/xONEIROS/MorphL2.git
cd MorphL2
```
2. نصب پیشنیاز ها
```
pip install web3 colorama
```

3. اجرا 

```
python3 main.py
```
